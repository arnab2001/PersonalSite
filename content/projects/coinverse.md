## Overview

Coinverse is a comprehensive cryptocurrency tracking platform that provides real-time prices, news, and historical data visualization. Selected for JWOC 2023 open source contribution program.

## Key Features

- **Live Crypto Prices**: All cryptocurrency prices at a glance on the home page
- **Crypto News Feed**: Latest news from the cryptocurrency world
- **Historical Charts**: Price history visualization with Chart.js
- **Coin-Specific News**: Search and get latest news on any specific cryptocurrency
- **Detailed Coin Information**: Origin, facts, and comprehensive data about each coin
- **Favorites System**: Bookmark your favorite cryptocurrencies for quick access

## Technical Implementation

Built with React and Redux for state management, using Ant Design for a polished UI. Integrates with Coinranking API for real-time cryptocurrency data and Bing News API for relevant news articles. Chart.js provides interactive price history visualizations.
