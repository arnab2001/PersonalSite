## Overview

Hype The Hike is a community-driven tourism platform that empowers local communities through Web3 technology. Winner of two tracks at Hack The Mountain 3.0 for innovative use of Lighthouse and Arcana.

## Key Features

- **Event Discovery**: Browse and join upcoming community tourism events
- **Arcana Authentication**: Secure, decentralized sign-in system
- **NFT Identity Proof**: Mint NFTs that serve as event identity verification
- **Activity Selection**: Choose preferred activities within events
- **IPFS Storage**: Images stored on IPFS using Lighthouse for decentralization
- **Polygon Network**: Fast, low-cost transactions on Polygon blockchain

## Technical Implementation

Built with Next.js and React for the frontend, styled with Chakra UI. Integrates Arcana Auth for decentralized authentication, Thirdweb SDK for NFT minting, and Lighthouse for IPFS storage. Deployed on Spheron for decentralized hosting.

## How It Works

1. Users discover upcoming tourism events on the landing page
2. Sign in using Arcana Auth for decentralized authentication
3. Select an event and choose preferred activities
4. Connect Metamask wallet to mint an NFT identity proof
5. NFT images stored on IPFS via Lighthouse
6. NFT serves as identity verification throughout the event
