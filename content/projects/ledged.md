## Overview

Ledged is a decentralized ledger management system that provides immutable, blockchain-based data storage. Winner of the "Best project built on 5ire" track at Diversion 2k23 hackathon.

## Key Features

- **Metamask Integration**: Secure wallet-based authentication
- **Custom Ledger Creation**: Create ledgers with custom fields and data structures
- **File Upload Support**: Attach documents and files to ledger entries
- **Preview System**: Review ledger data before deployment to blockchain
- **Ledger History**: View all previously created ledgers
- **Decentralized Storage**: Immutable records on the 5ire blockchain

## Technical Implementation

Built with React and Vite for a fast, modern development experience. Integrates with the 5ire blockchain for decentralized data storage, ensuring immutability and transparency. Metamask wallet integration provides secure user authentication.
