## Overview

CocktailBar is a clean, simple web application for discovering cocktail recipes. Built with vanilla JavaScript to demonstrate core web development fundamentals without frameworks.

## Key Features

- **Recipe Search**: Search for cocktails by name
- **Detailed Information**: View ingredients, measurements, and preparation instructions
- **Clean UI**: Simple, intuitive interface built with pure CSS
- **API Integration**: Real-time data from the Cocktail API

## Technical Implementation

Built entirely with vanilla HTML, CSS, and JavaScript. Demonstrates fundamental web development skills including DOM manipulation, API integration, and responsive design. Deployed on Netlify for fast, reliable hosting.
